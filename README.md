# GestorTerapeuticas
GestorOlimpíadas é um sistema em Java para gestão de competições olímpicas, desenvolvido com o objetivo de organizar e acompanhar todas as entidades envolvidas na realização de uma Olimpíada desportiva. Este projeto implementa funcionalidades essenciais de um sistema de gestão de eventos desportivos, incluindo o registo e gestão de modalidades, atletas, utilizadores e outros processos típicos de uma competição.

Funcionalidades principais:
- Gestão de modalidades desportivas
- Registo e acompanhamento de atletas
- Administração de utilizadores e permissões
- Fluxos comuns a sistemas de gestão de eventos
- Documentação e diagramas UML incluídos
